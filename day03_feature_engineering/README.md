# Day 3: Feature Engineering & Ensemble Learning

## Project Overview
This project implements an end-to-end Machine Learning pipeline on the Titanic dataset.

## Files
1. **feature_engineering.py**: Cleans data, imputes missing values, and scales features.
2. **dimensionality_reduction.py**: Visualizes data using PCA and t-SNE.
3. **ensemble_models.py**: Implements XGBoost and LightGBM.
4. **hyperparameter_tuning.py**: Optimizes Random Forest using Grid Search.
5. **ml_pipeline.py**: Automates the execution of all scripts.

## How to Run
Run the pipeline using:
python ml_pipeline.py
